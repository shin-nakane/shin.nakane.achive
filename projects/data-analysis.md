# 研究内容

## 

このプロジェクトでは、Pythonのデータ分析ライブラリ（Pandas, NumPy）と可視化ライブラリ（Matplotlib, Seaborn）を使用し、様々な環境変数（温度、湿度、日照時間）と特定の作物の生長データを統合的に分析しました。

**主な機能:**
* 時系列データのトレンド分析
* 変数間の相関解析
* 対話的なグラフ生成（Jupyter Notebook）
